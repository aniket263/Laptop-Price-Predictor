# Laptop Price Predictor

## Overview

The Laptop Price Predictor is a Machine Learning project that estimates the price of a laptop based on its specifications such as brand, processor, RAM, storage, and operating system.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## Features

* Predicts laptop prices
* Real-time price estimation
* User-friendly interface
* Machine Learning-based prediction

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Objective

To develop a machine learning model that predicts laptop prices using various hardware and software specifications.
